# rhhr

Solves rush hour puzzle. Initial state given by text file of the form:

```
    ....AA
    ..BBCC
    rr..EF
    GGHHEF
    ...IEF
    ...IJJ
```
useage:

```bash
    python interface.py -f ini.txt
```
